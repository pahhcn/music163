# 网易云音乐热门歌单数据分析系统

🎵 一个基于Python的网易云音乐热门歌单数据爬取、分析和可视化系统

## ✨ 功能特色

### 🕷️ 数据爬取
- **热门歌单爬取**: 支持分页爬取网易云音乐热门歌单
- **歌曲信息爬取**: 批量获取歌单内的歌曲详细信息
- **智能反爬**: 随机延时、User-Agent轮换等反爬策略
- **断点续爬**: 支持中断后继续爬取

### 📊 数据分析
- **歌单分析**: 播放量、收藏数、标签分布等多维度分析
- **歌曲分析**: 热度分布、时长统计、跨歌单热门歌曲
- **歌手分析**: 歌手排行、专辑热度、多维度能力评估
- **创建者分析**: 歌单创建者贡献度统计

### 🎨 数据可视化
- **17个交互式图表**: 涵盖歌单和歌曲的全方位可视化
- **炫酷界面设计**: 渐变背景、悬停动画、响应式布局
- **多种图表类型**: 柱状图、饼图、散点图、雷达图、词云等
- **导航式浏览**: 分页展示，支持键盘导航

## 🚀 快速开始

### 环境要求
- Python 3.7+
- 依赖包见 `requirements.txt`

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/pahhcn/music163.git
cd music163
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **运行程序**
```bash
python main.py
```

### 使用流程

1. **爬取歌单数据**
   - 运行程序，选择 `1 - 爬取热门歌单数据`
   - 设置爬取页数（每页50个歌单）

2. **爬取歌曲数据**
   - 选择 `2 - 爬取歌单内的歌曲`
   - 可选择爬取所有歌单或指定数量的歌单

3. **生成可视化报告**
   - 选择 `4 - 生成可视化报告`
   - 在浏览器中打开生成的HTML报告

## 🛠️ 技术栈

- **爬虫**: requests + 网易云音乐API
- **数据库**: SQLite
- **数据分析**: pandas + numpy
- **可视化**: pyecharts
- **日志**: Python logging
- **其他**: jieba (中文分词)

## 📊 数据说明

### 歌单数据字段
- `playlist_id`: 歌单ID
- `playlist_name`: 歌单名称
- `creator_name`: 创建者名称
- `play_count`: 播放次数
- `subscribed_count`: 收藏次数
- `track_count`: 歌曲数量
- `tags`: 标签
- `description`: 描述

### 歌曲数据字段
- `song_id`: 歌曲ID
- `song_name`: 歌曲名称
- `artist`: 歌手
- `album`: 专辑
- `duration`: 时长（毫秒）
- `popularity`: 热度值
- `playlist_id`: 所属歌单ID
